
# Comandos do Git

Comandos básicos para usar o git no computador.

| Comando | Descrição |
|---------|------------|
| cd | navegar entre pastas, mudar de pastas;  [cd ..] para voltar
|dir | listar pastas
| mkdir | criar pasta
